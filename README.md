## 🚀 Quick Start

### 📋 Prerequisites

```bash
✅ Flutter SDK (3.0+)
✅ Dart SDK  
✅ Android Studio / VS Code
✅ Android SDK
```

### ⚡ Installation

```bash
# 1️⃣ Clone repository
git clone https://github.com/username/modal-management-app.git
cd modal-management-app

# 2️⃣ Install dependencies
flutter pub get

# 3️⃣ Run the app
flutter run
```
